My Work

Horiseon

In this project, I worked on improving the code for a webpage of a marketing agency, Horiseon, so that it meets the accessibility standards. To accomplish the task, I updated and refactored the existing code, such that elements followed a logical structure independent of styling and positioning, the image elements had accessible 'alt' attributes, the heading attributes followed a sequetial order and the title was concise and descriptive.
